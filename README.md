# KAIdison 脑机接口知识胶囊系统

## 项目概述

**KAIdison** = KAI + DATM + Edison

基于**知识胶囊哲学**构建的脑机接口(BCI)知识管理系统，为天津大学 MBC 提供全面的技术路线管理、学术文献评估和案例知识库服务。

## 核心身份

### 数字科学家 (Digital Scientist)

践行知识胶囊哲学的AI系统，具备：

- **研究驱动** - 以探究和实证为核心
- **跨学科** - 具备多领域视野
- **历史意识** - 在时间长河中理解知识演变
- **涌现思维** - 关注复杂系统中的交互新属性

### 知识胶囊哲学 (Knowledge Capsule Philosophy)

四大支柱：

1. **封装 (Encapsulate)** - 将知识模块化、标准化
2. **溯源 (Traceability)** - 确保知识来源可追踪，保证真实性
3. **语义碰撞 (Semantic Collision)** - 允许不同概念之间产生关联和火花
4. **跨域融合 (Cross-domain Fusion)** - 打破学科壁垒，进行综合性思考

> **Core Belief:** "Knowledge is discovered, not created."  
> 知识是被发现的，而非被创造的。

## 核心特性

### 🧠 胶囊类型体系

| 胶囊类型 | 描述 | 应用场景 |
|---------|------|---------|
| **技术路线胶囊** | 侵入式/非侵入式/数字孪生的参数模型 | 技术选型、方案对比 |
| **论文胶囊** | 学术文献的 DATM 评分版 | 学术研究、文献调研 |
| **案例胶囊** | 手术/康复/伦理案例 | 临床参考、伦理决策 |
| **实验数据胶囊** | EEG/神经元信号数据 | 数据共享，分析研究 |
| **大学知识胶囊** | 天津大学、复旦大学研究成果 | 跨学科研究、学术合作 |

### 📊 DATM 评分体系

基于四大维度评估知识价值：

```
DATM = {
  Truth (科学性)       → 技术的科学依据 (0-100)
  Goodness (安全性)    → 安全性评估 (0-100)
  Beauty (美学)        → 用户体验/界面 (0-100)
  Intelligence (实用性) → 实用价值 (0-100)
}
```

### 🔗 知识图谱

- **技术路线关系网络** - 侵入式 ↔ 非侵入式 ↔ 数字孪生
- **机构关联网络** - 天津大学、复旦大学、中科院等
- **学者网络** - 领域专家、论文作者、项目参与者
- **语义碰撞引擎** - 跨域概念关联与知识涌现
